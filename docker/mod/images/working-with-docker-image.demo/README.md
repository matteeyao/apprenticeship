# Working w/ Docker Images

## About this lab

Time to put your Docker image skills to the test! In this lab, you will use your newfound knowledge of Docker images to pull, build, and launch containers.

Put your knowledge to the test by pulling the latest Alpine and CentOS images from Docker Hub.

## Learning objectives

[ ] Pull the latest `alpine` image.

[ ] Pull the latest `httpd` image.

[ ] Pull nginx 1.15.

[ ] Compare the history.
